Kurulum

5 Dakikada Başla

Python ve Node.js kurulu olması yeterli.

1
Repoyu klonla
GitHub'dan projeyi indir.
git clone https://github.com/gorkbul-svg/clauadeaias.git
cd REPO
2
Python bağımlılıklarını kur
Sanal ortam oluşturup gereksinimleri yükle.
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
3
API anahtarını ayarla
.env.example'ı kopyalayıp anahtarını gir.
cp .env.example .env
# .env dosyasını aç, ANTHROPIC_API_KEY'i doldur
4
Backend ve frontend'i başlat
İki ayrı terminalde çalıştır.
# Terminal 1 — Backend
uvicorn api:app --reload --port 8000

# Terminal 2 — Frontend
cd frontend && npm install && npm run dev
